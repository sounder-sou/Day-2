# Phishing Email Analysis

This project demonstrates the analysis of a phishing email using cybersecurity investigation techniques. The analysis includes sender verification, URL inspection, SPF/DMARC checks, SMTP testing, blacklist verification, and identification of phishing characteristics such as spoofed domains and urgent language.

## Tools Used
- MXToolbox
- Email Header Analysis
- Domain Reputation Checks

## Outcome
The investigated email was successfully identified as a phishing attempt based on multiple security indicators.
